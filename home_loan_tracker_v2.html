☁ Google Drive Setup

Your data syncs to a JSON file in your Google Drive. Uses Google OAuth — no backend, no passwords stored. One-time setup takes ~5 minutes.

1
Run python3 serve.py — open the app at http://localhost:8765 (OAuth requires localhost, not file://)
2
Go to Google Cloud Console → new project → enable Google Drive API → Credentials → OAuth 2.0 Client ID (Web App) → add http://localhost:8765 as authorised origin
3
Paste your Client ID into the HTML file where it says YOUR_CLIENT_ID_HERE
4
Click "Save to Drive" — a Google sign-in popup will appear. After signing in, your data syncs automatically on every entry.
